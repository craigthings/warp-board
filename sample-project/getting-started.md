# Getting Started

A quick guide to get up and running with the project.

---

## Prerequisites

Before you begin, make sure you have:

- Node.js 18 or higher installed
- npm or yarn package manager
- A code editor (VS Code recommended)

## Installation

1. Clone the repository
2. Run `npm install`
3. Start the dev server with `npm run dev`

## Next Steps

Once you have the basics running, check out the Architecture document to understand how everything fits together.
